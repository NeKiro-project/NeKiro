本模块是纯契约与校验库，按职责拆分为三个文件：
- `contracts.go`：集中声明所有跨进程消息的 Go 结构体（AgentCard、CatalogEntry、Installation、InvocationRecord、PlatformError 等）、全局常量（SchemaVersion、APIVersion、大小限制）以及类型别名（如 `PlatformError = PlatformErrorV2`、`A2AProfile = A2AProfileV02`），作为上层唯一可见的公共 API。
- `validate.go`：通过 `//go:embed schemas/*.json openapi/*.yaml a2a-profile/*.json ...` 将 JSON Schema、OpenAPI 文档与语义规则 Markdown 编译进二进制；使用 `github.com/santhosh-tekuri/jsonschema/v6` 在 `NewValidator()` 中注册 Draft2020、自定义 `semver-range` format，并编译出各 schema 实例；提供 `Validate*` / `Decode*` 方法链，先做 `DisallowUnknownFields` + `UseNumber` + `requireJSONEOF` 的严格反序列化，再调用 `validateMappedValue` 走 JSON Schema，最后叠加语义校验（如 `EvaluateAgentCardSemantics`、`validateInstallationV2Semantics`），返回 `SchemaValidationError` / `SemanticValidationError` 两类错误。
- `a2a_profile_v02.go`：实现 A2A Profile v0.2 的具体协议约束，内嵌 `a2a-profile/v0.3.0/` 下的 profile 与 conformance fixture，定义 `A2AConformanceRuleIDV02`、`A2AProtocolErrorV02` 等枚举及 `requiredA2AConformanceRules` / `isA2AConformanceRuleAllowed` 等规则判定函数，并通过 `LoadA2AProfileV02`、`LoadA2AConformanceManifestV02` 暴露给测试套件。
依赖方向单向：`a2a_profile_v02.go` → `contracts.go`（引用 `PlatformErrorCode`、`A2ASDK` 等），`validate.go` → `contracts.go`（使用全部结构体），三者均不反向依赖业务代码。
