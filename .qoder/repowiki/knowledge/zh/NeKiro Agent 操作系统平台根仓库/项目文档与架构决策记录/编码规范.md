- ADR 文件统一采用「# ADR NNNN: 标题」+ Status/Date 头部 + Context/Decision/Consequences 三段结构，并按四位数字递增命名。
- 交接文档 `CURRENT.md` 在每次更新时维护「Updated」时间戳、分支/基线 commit、PR 号与 Issue 链接，并以「Fallback delta: removed X, retained Y, added Z, net W」量化行为差异。
- 所有外部依赖（Git remote、上游仓库、分支名、required local Git identity）在 `CURRENT.md` 中显式声明，避免读者自行猜测。
- 命令示例统一使用 PowerShell 语法（`$env:`、`Copy-Item`、`Set-Location`），并在关键步骤后附加预期输出或健康检查命令。
