- 所有测试辅助函数均声明 `t.Helper()` 并使用 `t.Fatalf` 而非 panic，失败时附带完整响应体或 SQL 查询结果以便定位。
- JSON 解码一律使用 `json.NewDecoder(...).UseNumber()`，防止大数被转为 float64 导致精度丢失。
- HTTP 错误断言统一走 `assertPlatformError(t, result, status, code)`，同时比对 `PlatformError.Code` 与 `x-nek-trace-id` 响应头。
- 敏感材料（token、digest、域名、输入 schema 等）集中声明为包级常量，并通过 `assertLogsAreSecretSafe` 在 defer 中扫描 stdout/stderr 禁止泄露。
- 并发竞态用例通过 `sync.WaitGroup` + 无缓冲 channel + `close(start)` 触发 goroutine 同步启动，再统计各 goroutine 写入的状态码分布做最终断言。
- 对外部命令（go build、migrate、serve）的调用统一封装为 `runCommand` / `startServer`，通过 `environmentWith` 合并父进程环境与必要覆盖变量，避免直接拼接 os.Environ。
